# Przewodnik po Narzędziach Deweloperskich

Witamy w interaktywnym przewodniku po narzędziach używanych przez programistów! Ten repozytorium zawiera 6 ćwiczeń praktycznych, które krok po kroku wprowadzają w świat narzędzi deweloperskich.

## Dla kogo jest ten przewodnik?

Ten przewodnik jest skierowany do osób nietechnicznych, które chcą zrozumieć:
- Jak działają nowoczesne aplikacje
- Jakie narzędzia używają programiści
- Jak wygląda proces tworzenia i wdrażania aplikacji

## Wymagania

Przed rozpoczęciem upewnij się, że masz zainstalowane:
- [Node.js](https://nodejs.org/) (wersja 18 lub nowsza)
- [Docker](https://www.docker.com/) (dla ćwiczeń 5 i 6)

## Struktura Ćwiczeń

### 📁 Ćwiczenie 1: Podstawowy JavaScript
**Czego się nauczysz:** Jak uruchomić prosty skrypt JavaScript
- Zmienne, funkcje, wyświetlanie wyników
- Uruchamianie za pomocą Node.js

### 📁 Ćwiczenie 2: TypeScript
**Czego się nauczysz:** Dlaczego programiści używają TypeScript
- Typy danych i ich znaczenie
- Kompilacja kodu

### 📁 Ćwiczenie 3: NPM i Dependencies
**Czego się nauczysz:** Jak używać zewnętrznych bibliotek
- Czym jest NPM (Node Package Manager)
- Instalowanie i używanie zewnętrznych pakietów

### 📁 Ćwiczenie 4: NPM Scripts
**Czego się nauczysz:** Automatyzacja zadań
- Definiowanie skryptów w package.json
- Uruchamianie różnych zadań jedną komendą

### 📁 Ćwiczenie 5: Docker
**Czego się nauczysz:** Konteneryzacja aplikacji
- Czym są kontenery i dlaczego są ważne
- Tworzenie obrazu Docker

### 📁 Ćwiczenie 6: Docker Compose
**Czego się nauczysz:** Orkiestracja wielu usług
- Jak łączyć aplikację z bazą danych
- Uruchamianie całego środowiska jedną komendą

### 📁 Ćwiczenie 7: Backend z Hono i PostgreSQL
**Czego się nauczysz:** Nowoczesne API z najszybszym frameworkiem
- Hono Framework (ultra-fast)
- REST API z CRUD operacjami
- Połączenie z bazą danych PostgreSQL

### 📁 Ćwiczenie 8: Frontend React
**Czego się nauczysz:** Nowoczesny frontend
- React z TypeScript
- Vite jako build tool
- Łączenie z backend API
- Responsive design

### 📁 Ćwiczenie 9: Turborepo Full Stack
**Czego się nauczysz:** Monorepo architecture
- Turborepo dla zarządzania projektami
- Complete CRUD z DELETE functionality
- Professional development workflow
- Production-ready architecture

## Jak zacząć?

1. Sklonuj to repozytorium
2. Przejdź do folderu `exercise-1-javascript`
3. Przeczytaj instrukcje w README.md danego ćwiczenia
4. Wykonaj ćwiczenie
5. Przejdź do kolejnego!

## Pomoc

Jeśli napotkasz problemy:
1. Sprawdź README.md w folderze ćwiczenia
2. Upewnij się, że masz zainstalowane wszystkie wymagania
3. Sprawdź czy jesteś w odpowiednim folderze

## Cel Edukacyjny

Po ukończeniu wszystkich ćwiczeń będziesz rozumieć:
- Jak powstają nowoczesne aplikacje webowe
- Dlaczego programiści używają tych konkretnych narzędzi
- Jak wygląda typowy workflow deweloperski
- Jakie są korzyści z automatyzacji i konteneryzacji
- **Kompletny Full Stack development process**
- **Professional monorepo architecture**
- **Modern tech stack (React, Hono, PostgreSQL, Docker, Turborepo)**

## 🎯 Learning Path Summary

**Beginner → Intermediate → Advanced → Professional**

1-3: **Podstawy** (JavaScript, TypeScript, NPM)
4-6: **Narzędzia** (Scripts, Docker, Compose)
7-9: **Full Stack** (Backend, Frontend, Monorepo)
