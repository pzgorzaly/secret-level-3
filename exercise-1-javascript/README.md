# Ćwiczenie 1: Podstawowy JavaScript z Node.js

## Cel ćwiczenia
Poznać podstawy JavaScript i nauczyć się uruchamiać skrypty za pomocą Node.js.

## Czego się nauczysz?
- Czym są zmienne i funkcje
- Jak wyświetlać informacje w konsoli
- Jak uruchomić kod JavaScript poza przeglądarką

## Co znajduje się w tym folderze?
- `script.js` - główny plik z kodem JavaScript
- `README.md` - ten plik z instrukcjami

## Jak uruchomić ćwiczenie?

### Krok 1: Sprawdź czy masz Node.js
```bash
node --version
```

Jeśli widzisz numer wersji (np. v18.17.0) - masz Node.js!
Jeśli nie, zainstaluj go z [nodejs.org](https://nodejs.org/)

### Krok 2: Uruchom skrypt
```bash
node script.js
```

## Co się dzieje pod maską?

1. **Node.js** to środowisko uruchomieniowe JavaScript
   - Pozwala uruchamiać kod JavaScript poza przeglądarką
   - Używane do tworzenia serwerów, narzędzi, aplikacji

2. **JavaScript** to język programowania
   - Zmienne przechowują dane
   - Funkcje grupują kod w logiczne bloki
   - `console.log()` wyświetla informacje w terminalu

## Pytania do przemyślenia

1. Co by się stało, gdybyś zmienił wartość zmiennej `imie`?
2. Jak myślisz, dlaczego programiści używają funkcji?
3. Jakie inne informacje mogłyby być przydatne w aplikacji?
